# Python Learning Journey 🚀

Benvenuti nel mio repository dedicato alla formazione continua in Python. Questo spazio raccoglie la mia evoluzione tecnica attraverso esercizi pratici e approfondimenti teorici.

## 📂 Organizzazione dei Moduli
Il percorso è suddiviso in moduli progressivi:

* **Modulo 1: Fondamenti** - Gestione di variabili, tipi di dato e operatori logici (es: calcolo età e operazioni aritmetiche).
* **Modulo 2: Controllo di Flusso** - Input utente, manipolazione stringhe e istruzioni condizionali (es: verifica anni bisestili).
* **Modulo 3: Algoritmi su Stringhe** - Implementazione di logiche per la compressione del testo e ricerca di sottostringhe palindrome.
* **Modulo 4: Funzioni e Logica** - Definizione di funzioni (`def`), algoritmi di cifratura Morse e generazione di password.
* **Modulo 5: Programmazione Funzionale** - Utilizzo di `lambda functions`, chiusure (closures) e ricorsione.

## 🛠️ Competenze Tecniche
- **Linguaggio:** Python 3.x
- **Teoria:** Database SQL (DDL, indici), Espressioni Regolari (RegEx) per pattern matching.
- **Tools:** Git, GitHub, Gestione di licenze Open Source (MIT).
